Пример использования скрипта /bin/lua/vp_auto_2_lua.lua
=============================

Модуль для манипуляций с файлом '<vp_auto_progect>/out/*_для_заказа.txt' ( групповая ведомость покупных на изделие ).

Преобразует файл в таблицу lua ./out/DeviceBase.lua.
Файл открывается и редактируется простым редактром кода, загружается в память интерпретатором Lua. Удобная альтернатива
классическим базам данных. Подход проигрывает в скорости работы, но выигрывает в наглядности представления.


Данное преобразование призвано помочь при анализе баз с названиями элементов на предмет выявления некорректных записей.
