Пример использования скрипта /bin/lua/vp_auto_2_csv.lua
=============================

Модуль для манипуляций с файлом '<vp_auto_progect>/out/*_для_заказа.txt' ( групповая ведомость покупных на изделие ).

Преобразует файл в *.CSV (Comma-Separated Values) формат, открываемый в электронных таблицах типа Microsoft Excel и/или LibreOffice Calc

Данное преобразование призвано помочь при формировании заказа элементной базы.
