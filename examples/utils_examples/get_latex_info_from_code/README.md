Пример использования скрипта /bin/lua/get_latex_info_from_code.lua
=============================

Модуль для извлечения LaTeX - описаия из многострочных комментариев в коде.
Альтернатива Doxygen. Код содержит более удобоваримое описание. Собственно, что автор напишет в комментариях и как это напишет - это и пойдёт в документ.
