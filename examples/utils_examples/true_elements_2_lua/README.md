Пример использования скрипта /bin/lua/true_elements_2_lua.lua
=============================

Модуль для манипуляций с файлом '/res/true_elements.txt' (файл с одобренными наименованиями элементов и их истроией переименований).

Преобразует файл в таблицу lua ./out/elements_base_true.lua.
Файл открывается и редактируется простым редактром кода, загружается в память интерпретатором Lua. Удобная альтернатива
классическим базам данных. Подход проигрывает в скорости работы, но выигрывает в наглядности представления.

Для корректной работы нужен файл ../names_2_lua/out/names.lua (нужно запустить сначала  ../names_2_lua/runme.bat).

Данное преобразование призвано помочь при анализе баз с названиями элементов на предмет выявления некорректных записей.
