Пример использования скрипта /bin/lua/names_2_lua.lua
=============================

Модуль для манипуляций с файлом '/res/names.txt' (файл с названиями элементов в единственном/множественном числе).

Преобразует файл в таблицу lua ./out/names.lua


Данное преобразование призвано помочь при анализе баз с названиями элементов на предмет выявления некорректных записей.
